# 이미지 기반 미로 탐색

미로 이미지를 컴퓨터 비전으로 분석하고 BFS로 최단 경로를 구한 뒤 결과를 다시 이미지에 표시하는 프로젝트입니다.

## 주요 기능

- OpenCV로 이미지를 흑백 변환하고 미로 영역을 추출합니다.
- 이진화와 형태학 연산으로 벽과 이동 가능한 길을 구분합니다.
- 가장자리의 연결된 픽셀을 묶어 시작점과 도착점을 찾습니다.
- BFS로 최단 경로를 탐색하고 부모 좌표를 따라 실제 경로를 복원합니다.
- 시작점, 도착점과 이동 경로를 결과 이미지에 그려 저장합니다.

## 기술

`Python` · `OpenCV` · `NumPy` · `BFS` · `Image Processing`
