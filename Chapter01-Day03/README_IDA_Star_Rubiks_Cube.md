# IDA* 루빅스 큐브 풀이

3×3 루빅스 큐브의 상태와 회전을 배열로 모델링하고 IDA* 탐색으로 섞인 상태의 해법을 찾는 알고리즘 프로젝트입니다.

## 주요 기능

- 큐브의 6개 면과 각 면의 3×3 스티커를 다차원 리스트로 표현합니다.
- `U`, `D`, `F`, `B`, `L`, `R` 회전에 따라 인접 면의 행과 열을 함께 이동합니다.
- 무작위 회전으로 시작 상태를 만들고 완성 상태와 비교합니다.
- 휴리스틱으로 남은 비용을 추정하고 깊이 제한 DFS를 반복하는 IDA* 탐색을 수행합니다.
- 찾은 회전 순서와 탐색한 상태 수를 출력해 결과를 확인합니다.

## 기술

`Python` · `IDA*` · `Heuristic Search` · `Depth-first Search` · `Simulation`

## 관련 파일

- [Python 소스](Cube.py)

## 배운 점

현실의 퍼즐 상태를 자료구조로 정확히 표현하는 과정과 휴리스틱을 활용해 탐색 공간을 줄이는 방법을 함께 경험했습니다.
